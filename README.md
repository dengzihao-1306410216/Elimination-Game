# Elimination-Game
项目名称：卡牌消除小游戏
项目用途：休闲娱乐，益智类的网页版小游戏
游戏玩法：所有卡牌盖着，然后任意翻开两张，若两张卡牌完全相同，则消除，接着翻开剩下的卡牌；
          若两张卡牌不同，则卡牌立即改过去，接着重新翻牌；
          知道所有卡牌消除完，则游戏结束！
技术：     HTML/CSS/jQuery
